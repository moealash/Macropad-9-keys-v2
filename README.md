<h2>
Explanation 
  
</h2>

<h4>
This is a Macroboard that has 9 keys to get around my pc faster with macros. I made it because I want to see if I have what it takes to have a future with electronics.
</h4>
<h2>
  
</h2>
<h2>Parts</h2>

<h3>

- **XIAO RP2040** It is the Usb type C connnecter
  
- **9 Cherry MX switches** It sends an input to the PCB
  
- **9 keycaps** It is a cover for the switchs
  
- **Custom 3d Case** It holds and protects the PCB
  
- **Custom PCB** It connects the wires to the electronics
</h3>
<h2>
  
</h2>
